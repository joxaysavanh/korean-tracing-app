# 🇰🇷 Korean Tracing App

A mobile application developed with Flutter to help users practice writing Korean characters (Hangul). This app provides an interactive canvas to trace vowels and consonants, along with audio pronunciation to enhance learning.

แอปพลิเคชันมือถือที่พัฒนาด้วย Flutter เพื่อช่วยให้ผู้ใช้ฝึกเขียนตัวอักษรเกาหลี (ฮันกึล) แอปนี้มาพร้อมกับกระดานวาดเขียนแบบโต้ตอบสำหรับฝึกคัดลายมือ ทั้งสระและพยัญชนะ พร้อมระบบเสียงอ่านเพื่อเสริมสร้างการเรียนรู้

---

## 🚀 Features | ฟีเจอร์หลัก

*   **Interactive Tracing:** Practice writing Korean vowels and consonants on an interactive canvas.
    *   ฝึกคัดลายมือ: ฝึกเขียนสระและพยัญชนะเกาหลีบนกระดานวาดเขียนแบบโต้ตอบ
*   **Step-by-Step Guidance:** Follow stroke order with visual guides and arrows.
    *   คำแนะนำทีละขั้นตอน: เรียนรู้ลำดับการเขียนที่ถูกต้องด้วยไกด์ไลน์และลูกศรนำทาง
*   **Text-to-Speech (TTS):** Hear the correct pronunciation of each character.
    *   ระบบเสียงอ่าน: ฟังการออกเสียงที่ถูกต้องของตัวอักษรแต่ละตัว
*   **Categorized Learning:** Separate tabs for Vowels and Consonants for easy navigation.
    *   การเรียนรู้แบบหมวดหมู่: แยกแท็บสระและพยัญชนะ เพื่อการใช้งานที่สะดวก

## 🛠️ Tech Stack | เทคโนโลยีที่ใช้

*   **Framework:** Flutter

## 📱 How to Run | วิธีการใช้งาน

1. Clone this repository to your local machine.
   * ทำการ Clone repository นี้ลงในเครื่องของคุณ
2. Make sure you have Flutter installed.
   * ตรวจสอบว่าได้ติดตั้ง Flutter เรียบร้อยแล้ว
3. Run `flutter pub get` to install dependencies.
   * รันคำสั่ง `flutter pub get` เพื่อติดตั้งไลบรารีที่จำเป็น
4. Run `flutter run` to start the app on your emulator or real device.
   * รันคำสั่ง `flutter run` เพื่อเริ่มแอปบน Emulator หรือเครื่องจริงของคุณ
